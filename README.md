## Các thành viên:
1) Nguyễn Thanh Tùng
2) Ngô Duy Khang
3) Lê Duy Thắng

## Nội dung

App Flutter công thức nấu ăn sử dụng SQLite, bao gồm các chức năng:

+ Xem món ăn
+ Yêu thích món ăn
+ Tìm kiếm món ăn
+ Xem món ăn theo danh mục
+ Hiện lịch sử các món đã xem

- Lưu ý: pub get trước khi chạy bài lần đầu tiên.
